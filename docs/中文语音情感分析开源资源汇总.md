# 中文语音情感分析开源资源汇总

> 更新时间: 2026年4月

---

## 一、GitHub 开源项目

### 1. 综合语音情感识别项目

#### Renovamen/Speech-Emotion-Recognition ⭐1300+
- **链接**: https://github.com/Renovamen/Speech-Emotion-Recognition
- **描述**: 语音情感识别 (支持中文文档)
- **技术栈**: Python, Keras, LSTM, CNN, SVM, MLP
- **数据集**: IEMOCAP, EMODB
- **特点**: 包含多种模型实现，有中文文档说明
- **语言**: Python

#### Vincent-ZHQ/CA-MSER ⭐160+
- **链接**: https://github.com/Vincent-ZHQ/CA-MSER
- **描述**: 基于协同注意力的多级声学信息语音情感识别
- **技术栈**: Python, PyTorch
- **论文**: ICASSP 2022
- **数据集**: IEMOCAP
- **语言**: Python

### 2. 语音/音频特征提取工具

#### librosa ⭐7500+
- **链接**: https://github.com/librosa/librosa
- **描述**: Python音频信号处理库
- **特点**: 支持MFCC、梅尔频谱等特征提取，是语音情感分析的基础工具
- **语言**: Python

#### openSMILE ⭐800+
- **链接**: https://github.com/audeering/opensmile-python
- **描述**: 开源特征提取工具，支持6000+音频特征
- **特点**: 广泛用于语音情感分析，支持eGeMAPS等特征集
- **语言**: Python/C++

#### wav2vec 2.0 / HuBERT (HuggingFace)
- **链接**: https://huggingface.co/models?pipeline_tag=audio-classification
- **描述**: 预训练语音模型，可用于情感分类
- **特点**: 支持微调，有大量预训练模型

## 二、中文语音情感数据集

### 1. CASIA 中文情感语音数据集
- **来源**: 中国科学院自动化研究所
- **语言**: 中文
- **情感类别**: 高兴、愤怒、悲伤、恐惧、中性等
- **获取方式**: 学术研究用途可申请

### 2. 汉语情感语音数据库 (SEU)
- **来源**: 东南大学
- **语言**: 汉语
- **特点**: 专业中文语音情感数据集

### 3. IEMOCAP (多语言)
- **来源**: USC
- **语言**: 英语为主，可迁移到中文
- **特点**: 最常用的语音情感数据集

## 四、Python 库推荐

### 1. speechbrain ⭐3500+
```bash
pip install speechbrain
```
- **链接**: https://github.com/speechbrain/speechbrain
- **功能**: 端到端语音处理工具包
- **特点**: 支持情感识别、说话人识别等任务

### 2. Transformers (HuggingFace)
```bash
pip install transformers
```
- **链接**: https://huggingface.co/
- **功能**: 预训练模型，支持音频分类
- **中文支持**: 可通过微调支持中文

### 3. pyAudioAnalysis
```bash
pip install pyAudioAnalysis
```
- **链接**: https://github.com/tyiannak/pyAudioAnalysis
- **功能**: 音频特征提取和分类

### 4. emovoz
- **链接**: https://github.com/MachineLN/emovoz

- **语言**: Python

## 五、推荐方案

### 快速上手方案
1. **使用百度/阿里云API** - 最快实现中文语音情感分析
2. **使用预训练模型微调** - HuggingFace上搜索audio classification模型

### 自研方案
1. **特征提取**: librosa + openSMILE
2. **模型训练**: PyTorch/TensorFlow
3. **数据集**: CASIA或自建数据集

### 开源项目推荐优先级
1. Renovamen/Speech-Emotion-Recognition (文档齐全，有中文支持)
2. speechbrain (功能全面)
3. CA-MSER (学术论文支持)

## 六、注意事项

1. **中文语音情感分析的特殊性**:
   - 声调信息对情感识别很重要
   - 需要针对中文优化的特征提取
   
2. **数据集问题**:
   - 高质量中文情感语音数据集较少
   - 可以考虑数据增强或跨语言迁移

3. **API选择**:
   - 商业API成本较高但效果好
   - 开源方案需要自行调优

# 快速开始指南

## 在 IntelliJ IDEA 中运行项目

### 方法一：使用 IDEA 的 Maven 工具（推荐）

1. **打开项目**
   - 启动 IntelliJ IDEA
   - 选择 `File` -> `Open`
   - 选择项目目录 `C:/Users/Administrator/IdeaProjects/InterviewSimulation`

2. **等待 Maven 导入**
   - IDEA 会自动检测 Maven 项目并导入依赖
   - 等待右下角进度条完成

3. **编译项目**
   - 在右侧工具栏找到 `Maven` 标签
   - 展开 `Lifecycle`
   - 双击 `compile` 进行编译
   - 或者双击 `package` 进行打包

4. **运行应用**
   - 打开 `src/main/java/com/ruijie/interview/Application.java`
   - 点击文件左侧的绿色三角形运行按钮
   - 或者右键选择 `Run 'Application.main()'`

5. **访问应用**
   - 打开浏览器访问：`http://localhost:8080/api/`

### 方法二：使用 IDEA 终端

1. **打开终端**
   - 在 IDEA 底部点击 `Terminal` 标签

2. **执行 Maven 命令**
   ```bash
   # 编译
   mvnw.cmd clean compile -DskipTests
   
   # 运行
   mvnw.cmd spring-boot:run
   ```

### 配置 API Key

在运行之前，需要配置大模型 API Key：

**方式一：环境变量（推荐）**
```cmd
# Windows CMD
set LLM_API_KEY=your-api-key-here

# Windows PowerShell
$env:LLM_API_KEY="your-api-key-here"
```

**方式二：修改配置文件**
编辑 `src/main/resources/application.yml`：
```yaml
app:
  llm:
    api-key: your-actual-api-key-here
```

## 获取 API Key

1. 访问 [OpenRouter](https://openrouter.ai/)
2. 注册/登录账号
3. 点击 `API Keys` 创建新的 Key
4. 复制 Key 并配置到项目中

## 首次运行

首次运行时会自动：
- 创建 SQLite 数据库文件 (`data/interview.db`)
- 初始化岗位数据（后端、前端、测试、算法工程师）
- 初始化题库数据
- 创建知识库文件 (`data/knowledge-base/*.json`)

## 测试功能

1. **注册账号**
   - 访问首页，点击 `立即注册`
   - 输入用户名、密码、姓名
   - 点击注册

2. **选择岗位**
   - 登录后进入岗位选择页面
   - 点击任意岗位卡片

3. **开始面试**
   - 切换到 `开始面试` 标签
   - 点击岗位开始面试
   - 回答每个问题

4. **查看报告**
   - 完成所有问题后查看评估报告
   - 包含多维度评分和改进建议

## 常见问题

### 端口被占用
修改 `application.yml`：
```yaml
server:
  port: 8081
```

### 依赖下载失败
- 检查网络连接
- 在 IDEA 中：`File` -> `Settings` -> `Build, Execution, Deployment` -> `Maven` -> `Importing`
- 勾选 `Always update snapshots`

### 中文乱码
确保文件编码为 UTF-8：
- `File` -> `Settings` -> `Editor` -> `File Encodings`
- 全部设置为 `UTF-8`

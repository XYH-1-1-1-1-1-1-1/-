# 快速开始指南

## 在 IntelliJ IDEA 中运行项目

### 方法一：使用 IDEA 的 Maven 工具（推荐）

1. **打开项目**
   - 启动 IntelliJ IDEA
   - 选择 `File` -> `Open`
   - 选择项目目录

2. **等待 Maven 导入**
   - IDEA 会自动检测 Maven 项目并导入依赖
   - 等待右下角进度条完成

3. **编译项目**
   - 在右侧工具栏找到 `Maven` 标签
   - 展开 `Lifecycle`
   - 双击 `compile` 进行编译
   - 或者双击 `package` 进行打包

4. **运行应用**
   - 打开 `src/main/java/com/ruijie/interview/Application.java`
   - 点击文件左侧的绿色三角形运行按钮
   - 或者右键选择 `Run 'Application.main()'`

5. **访问应用**
   - 打开浏览器访问：`http://localhost:8080/api/`

### 方法二：使用 IDEA 终端

1. **打开终端**
   - 在 IDEA 底部点击 `Terminal` 标签

2. **执行 Maven 命令**
   ```bash
   # 编译
   mvnw.cmd clean compile -DskipTests
   
   # 运行
   mvnw.cmd spring-boot:run
   ```

### 配置 API Key

在运行之前，需要配置阿里云 DashScope API Key：

**方式一：修改配置文件（推荐）**

编辑 `src/main/resources/application.yml`，将 `api-key` 替换为您的实际 API Key：

```yaml
app:
  llm:
    api-key: sk-your-actual-api-key-here  # 替换为您的 API Key
```

**方式二：环境变量**
```cmd
# Windows CMD
set LLM_API_KEY=sk-your-api-key-here

# Windows PowerShell
$env:LLM_API_KEY="sk-your-api-key-here"
```

## 获取 API Key

1. 访问 [阿里云 DashScope 控制台](https://dashscope.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 点击 `API Key 管理` 创建新的 Key
4. 复制 Key 并配置到项目中

## 首次运行

首次运行时会自动：
- 创建 SQLite 数据库文件 (`data/interview.db`)
- 初始化岗位数据（后端、前端、测试、算法工程师）
- 初始化题库数据
- 创建知识库文件 (`data/knowledge-base/*.json`)
- 创建简历存储目录 (`data/resumes/`)

   - 访问首页，点击 `立即注册`

## 常见问题

### 端口被占用
修改 `application.yml`：
```yaml
server:
  port: 8081  # 改为其他端口
```

### 依赖下载失败
- 检查网络连接
- 配置 Maven 镜像（阿里云镜像）
- 在 IDEA 中：`File` -> `Settings` -> `Build, Execution, Deployment` -> `Maven` -> `Importing`

### 中文乱码
确保文件编码为 UTF-8：
- `File` -> `Settings` -> `Editor` -> `File Encodings`
- 全部设置为 `UTF-8`

### API Key 无效
- 检查 API Key 是否正确复制
- 确认阿里云账号状态正常
- 检查是否有足够的 tokens 额度

### 语音输入不可用
- 使用 Chrome、Edge 等支持 Web Speech API 的浏览器
- 允许浏览器访问麦克风权限
- 确保使用 localhost 或 HTTPS 访问

### PDF 解析失败
- 确保 PDF 文件是文本型 PDF（非扫描图片）
- 如果是扫描版 PDF，需要配置 Tesseract OCR
- 检查 `tessdata` 目录是否存在

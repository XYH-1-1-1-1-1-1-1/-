# 语音情绪分析功能说明

## 功能概述

本系统为"真实面试"模式添加了语音情绪分析功能，使用阿里云 DashScope（通义千问）多模态大语言模型对面试者的语音回答进行情绪识别和评分。

## 情绪识别类型

系统可识别以下情绪状态：
- **CONFIDENT**（自信）- 面试者表现出自信的状态
- **NERVOUS**（紧张）- 面试者表现出紧张的状态
- **CALM**（从容）- 面试者表现得从容不迫
- **ANXIOUS**（焦虑）- 面试者表现出焦虑情绪
- **EXCITED**（兴奋）- 面试者表现得兴奋
- **NORMAL**（平静）- 面试者情绪平稳
- **FOCUSED**（专注）- 面试者注意力集中

## API 接口说明

### 1. 音频分析接口

**接口地址**: `POST /api/audio/analyze`

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| audio | File | 是 | 音频文件（支持 WAV、MP3、OGG、M4A 格式） |
| question | String | 否 | 面试问题（用于上下文分析） |
| transcript | String | 否 | 语音转录文本（如果前端已转录） |

**响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "transcript": "我认为 HashMap 是基于哈希表实现的...",
    "emotion": "CONFIDENT",
    "emotionAnalysis": "面试者回答问题时语速平稳，表达清晰，展现出对知识点的熟练掌握...",
    "confidenceLevel": 4,
    "technicalScore": 85,
    "communicationScore": 80,
    "logicScore": 78,
    "knowledgeDepth": 82,
    "overallScore": 81,
    "evaluationComment": "回答表现出色，技术点讲解清晰",
    "strengths": "技术基础扎实，表达流畅",
    "weaknesses": "可以进一步深入讲解底层原理"
  }
}
```

### 2. Base64 音频分析接口

**接口地址**: `POST /api/audio/analyze/base64`

**请求参数** (JSON 格式):
```json
{
  "audioBase64": "data:audio/wav;base64,...",
  "question": "请解释 HashMap 的实现原理",
  "transcript": "我认为 HashMap 是基于哈希表实现的..."
}
```

## 面试模式差异

| 功能 | 练习模式 (PRACTICE) | 真实面试 (REAL) |
|------|---------------------|-----------------|
| 输入方式 | 文字/语音可选 | 仅语音 |
| 情绪分析 | ❌ 不分析 | ✅ 分析 |
| 自信程度评估 | ❌ | ✅ |
| 回答可修改 | ✅ | ❌ |
| 评估报告 | 基础评分 | 包含情绪分析 |

## 评估报告展示

在真实面试模式结束后，评估报告会额外显示：

1. **情绪状态**: 使用 Emoji 图标和文字显示识别出的情绪
2. **情绪分析详情**: 详细说明情绪判断依据
3. **自信程度**: 使用星级（1-5 星）展示

## 配置说明

### application.yml 配置

```yaml
app:
  llm:
    # API 提供商
    provider: dashscope
    # API Key（需要替换为您的实际 Key）
    api-key: sk-xxxxxxxx
    # API 基础 URL
    base-url: https://dashscope.aliyuncs.com/compatible-mode/v1
    # 默认模型
    model: qwen-max
```

### 获取 API Key

1. 访问阿里云 DashScope 控制台：https://dashscope.console.aliyun.com/
2. 注册/登录阿里云账号
3. 创建 API Key
4. 将 API Key 复制到 application.yml 配置文件中

## 使用流程

1. **选择岗位和面试模式**
   - 在岗位选择页面选择要面试的岗位
   - 选择"真实面试"模式

2. **开始面试**
   - 点击岗位卡片开始面试
   - AI 面试官会逐题提问

3. **语音回答**
   - 点击"🎤 开始录音"按钮
   - 对着麦克风回答问题
   - 点击"⏹ 停止录音"结束

4. **查看评估报告**
   - 面试结束后自动生成评估报告
   - 报告包含情绪分析结果

## 注意事项

1. 需要浏览器支持 Web Speech API（推荐使用 Chrome、Edge 等现代浏览器）
2. 首次使用需要允许浏览器访问麦克风权限
3. 确保网络连接正常，以便调用云端大模型
4. 录音时请保持环境安静，以获得更好的识别效果

## 扩展建议

1. **语音特征分析**: 可增加语速、停顿、音量等语音特征分析
2. **面部表情识别**: 结合摄像头进行面部表情分析（需要额外权限）
3. **实时反馈**: 在面试过程中实时提供情绪反馈
4. **历史趋势**: 分析用户多次面试的情绪变化趋势

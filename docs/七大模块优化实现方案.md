## 二、打分逻辑模块优化

### 2.1 目标

突破传统单一分数评价方式，构建多维度、动态权重的评分体系。

### 2.2 评分维度设计

#### 2.2.1 五维评分模型

```
┌────────────────────────────────────────────────────────┐
│                    综合评分                              │
│                         ↑                               │
│     ┌───────────┬───────────┬───────────┬───────────┐  │
│     │ 技术正确性 │ 知识深度  │ 逻辑严谨性 │ 表达能力  │  │
│     │  (40%)    │  (30%)    │  (20%)    │  (10%)   │  │
│     └───────────┴───────────┴───────────┴───────────┘  │
│                         ↑                               │
│                  岗位匹配度 (加权系数)                    │
└────────────────────────────────────────────────────────┘
```

#### 2.2.2 各维度评分标准

**技术正确性 (Technical Correctness)**
- 评分依据：回答的技术准确性、概念正确性
- 大模型评估要点：
  - 关键词匹配度（如 HashMap 回答中包含"数组 + 链表 + 红黑树"）
  - 技术事实正确性（无错误陈述）
  - 完整性（覆盖核心知识点）
- 分数段：
  - 90-100：技术准确，无错误，覆盖全面
  - 70-89：基本正确，有小遗漏
  - 50-69：部分正确，有明显遗漏
  - 30-49：有较多错误
  - 0-29：几乎完全错误

**知识深度 (Knowledge Depth)**
- 评分依据：技术层次分析（基础/进阶/专家级）
- 层次定义：
  ```
  基础层 (1-3 分)：知道"是什么"
  进阶层 (4-7 分)：理解"为什么"
  专家层 (8-10 分)：能够"怎么用更好"
  ```
- 关联知识广度：能否关联相关知识点
- 示例：
  - 基础回答：HashMap 是哈希表实现
  - 进阶回答：HashMap 使用数组 + 链表 + 红黑树，负载因子 0.75
  - 专家回答：HashMap 扩容机制、红黑树转换条件、与 ConcurrentHashMap 对比

**逻辑严谨性 (Logic Rigor)**
- 评估要点：
  - 结构化程度（有总分总、分点论述）
  - 因果关系清晰度
  - 论述连贯性
- 大模型 Prompt 设计：
  ```
  请评估以下回答的逻辑结构：
  1. 是否有清晰的论述框架
  2. 因果关系是否成立
  3. 是否存在逻辑跳跃
  4. 论述是否连贯
  ```

**表达能力 (Expression Ability)**
- 结合语音分析结果：
  - 语速评估（过快/适中/过慢）
  - 停顿模式（思考性停顿 vs 紧张性停顿）
  - 用词准确性
- 文字表达评估：
  - 语句通顺度
  - 专业术语使用准确性
  - 简洁性

**岗位匹配度 (Position Matching)**
- 根据目标岗位 JD 要求评估
- 技能点覆盖程度
- 项目经验相关性

#### 2.3.3 面试轮次权重调整

随着面试轮次增加，逐步提高对软技能的评分权重：

```java
public class WeightCalculator {
    
    /**
     * 计算动态权重
     * @param baseWeights 基础权重
     * @param currentRound 当前轮次
     * @param totalRounds 总轮次
     * @return 调整后的权重
     */
    public static Map<String, Double> calculateDynamicWeights(
            Map<String, Double> baseWeights,
            int currentRound,
            int totalRounds) {
        
        Map<String, Double> adjusted = new HashMap<>(baseWeights);
        
        // 随着轮次增加，软技能权重逐步提升
        double progressRatio = (double) currentRound / totalRounds;
        
        // 表达能力权重随轮次增加（最多增加 50%）
        double expressionBoost = adjusted.get("expression") * (1 + progressRatio * 0.5);
        adjusted.put("expression", Math.min(expressionBoost, 0.20));
        
        // 逻辑严谨性权重随轮次增加（最多增加 25%）
        double logicBoost = adjusted.get("logic") * (1 + progressRatio * 0.25);
        adjusted.put("logic", Math.min(logicBoost, 0.30));
        
        // 相应降低技术权重，保持总和为 1
        double totalWeight = adjusted.values().stream().mapToDouble(Double::doubleValue).sum();
        double technicalReduction = adjusted.get("technical") * (totalWeight - 1.0);
        adjusted.put("technical", Math.max(adjusted.get("technical") - technicalReduction, 0.25));
        
        return adjusted;
    }
}
```

#### 2.4.2 对比评分算法

```java
@Service
public class ComparativeScoringService {
    
    @Autowired
    private AnswerSampleRepository sampleRepository;
    
    @Autowired
    private LlmService llmService;
    
    /**
     * 对比学习评分
     */
    public ComparativeScoreResult scoreWithComparison(
            String positionCode,
            String question,
            String userAnswer) {
        
        // 1. 检索相似问题的历史样本
        List<AnswerSample> samples = sampleRepository.findSimilarSamples(
            positionCode, question, 3);
        
        // 2. 构建对比学习 Prompt
        String prompt = buildComparativePrompt(userAnswer, samples);
        
        // 3. 调用大模型进行对比评分
        String result = llmService.simpleChat(prompt, buildSystemPrompt());
        
        // 4. 解析结果
        return parseComparativeScore(result);
    }
    
    private String buildComparativePrompt(String userAnswer, List<AnswerSample> samples) {
        StringBuilder sb = new StringBuilder();
        sb.append("【用户回答】\n").append(userAnswer).append("\n\n");
        sb.append("【参考样本】\n");
        
        for (AnswerSample sample : samples) {
            sb.append(String.format("--- %s 样本 ---\n", 
                sample.getSampleLevel().equals("excellent") ? "优秀" : 
                sample.getSampleLevel().equals("average") ? "普通" : "较差"));
            sb.append("回答：").append(sample.getAnswerContent()).append("\n");
            sb.append("评分：技术=").append(sample.getTechnicalScore())
              .append(", 深度=").append(sample.getDepthScore())
              .append(", 逻辑=").append(sample.getLogicScore()).append("\n\n");
        }
        
        sb.append("请对比用户回答与参考样本，给出评分和具体改进建议。");
        return sb.toString();
    }
}
```

#### 2.5.3 进步轨迹可视化

```java
// 新增服务：ProgressTrackingService
@Service
public class ProgressTrackingService {
    
    @Autowired
    private EvaluationReportRepository reportRepository;
    
    /**
     * 获取用户能力进步轨迹
     */
    public ProgressTrend getProgressTrend(Long userId, String dimension, int days) {
        LocalDateTime startDate = LocalDateTime.now().minusDays(days);
        List<EvaluationReport> reports = reportRepository.findByUserIdAndCreatedAtAfter(
            userId, startDate);
        
        List<ProgressPoint> points = reports.stream()
            .map(report -> new ProgressPoint(
                report.getCreatedAt(),
                extractDimensionScore(report, dimension)
            ))
            .collect(Collectors.toList());
        
        return new ProgressTrend(dimension, points, calculateTrend(points));
    }
    
    private String calculateTrend(List<ProgressPoint> points) {
        if (points.size() < 2) return "insufficient_data";
        
        double firstHalfAvg = points.subList(0, points.size()/2).stream()
            .mapToDouble(p -> p.getScore()).average().orElse(0);
        double secondHalfAvg = points.subList(points.size()/2, points.size()).stream()
            .mapToDouble(p -> p.getScore()).average().orElse(0);
        
        double change = (secondHalfAvg - firstHalfAvg) / firstHalfAvg * 100;
        
        if (change > 10) return "improving_fast";
        if (change > 0) return "improving";
        if (change > -10) return "stable";
        return "declining";
    }
}
```

### 3.3 检索优化策略

#### 3.3.1 多模态检索

```java
@Service
public class MultiModalRagService {
    
    @Autowired
    private RagService ragService;
    
    /**
     * 多模态检索：结合文本、语音特征
     */
    public List<KnowledgeItem> retrieveWithMultiModal(
            String positionId,
            String textQuery,
            VoiceFeatures voiceFeatures) {
        
        // 1. 文本检索
        List<KnowledgeItem> textResults = ragService.retrieveKnowledge(
            positionId, textQuery, 10);
        
        // 2. 根据语音特征调整检索策略
        if (voiceFeatures.isUncertain()) {
            // 用户语气不确定，检索更基础的知识
            List<KnowledgeItem> basicResults = retrieveByDifficulty(
                positionId, textQuery, 1, 3);
            textResults.addAll(basicResults);
        }
        
        if (voiceFeatures.isConfident()) {
            // 用户语气自信，检索更高阶的知识
            List<KnowledgeItem> advancedResults = retrieveByDifficulty(
                positionId, textQuery, 4, 5);
            textResults.addAll(advancedResults);
        }
        
        // 3. 去重并重新排序
        return deduplicateAndRerank(textResults, textQuery, voiceFeatures);
    }
}

    private double toneVariance;    // 音调变化

}
```

```java
@Service
public class TemporalAwareRagService {
    
    /**
     * 时效性感知的检索
     */
    public List<KnowledgeItem> retrieveWithTemporalAwareness(
            String positionId,
            String query) {
        
        List<KnowledgeItem> results = ragService.retrieveKnowledge(
            positionId, query, 20);
        
        // 对技术类知识设置时效权重
        results.forEach(item -> {
            double temporalWeight = calculateTemporalWeight(item);
            item.setScore(item.getScore() * temporalWeight);
        });
        
        // 重新排序
        results.sort(Comparator.comparingDouble(KnowledgeItem::getScore).reversed());
        
        return results.subList(0, Math.min(10, results.size()));
    }
    
    /**
     * 计算时效权重
     */
    private double calculateTemporalWeight(KnowledgeItem item) {
        // 技术更新快的领域（如前端框架），时效性更重要
        if (item.getCategory().contains("前端") || 
            item.getCategory().contains("框架")) {
            
            long daysSinceUpdate = ChronoUnit.DAYS.between(
                item.getUpdatedAt(), LocalDateTime.now());
            
            if (daysSinceUpdate > 365) return 0.7;  // 超过 1 年，权重降低
            if (daysSinceUpdate > 180) return 0.85;
            if (daysSinceUpdate > 90) return 0.95;
            return 1.0;
        }
        
        // 基础知识（如数据结构、算法）时效性影响较小
        return 1.0;
    }
}
```

```java
public class ProgressPredictionService {
    
    /**
     * 基于历史数据预测达到目标岗位要求的时间
     */
    public ProgressPrediction predictProgress(
            Long userId,
            String targetPosition,
            int targetScore) {

        PerformanceTrend trend = performanceTrendService.calculateTrend(
            userId, TrendPeriod.WEEKLY);
        
        // 计算当前水平

        // 计算进步速率（分/周）
        double improvementRate = trend.getChangeRate();
        
        // 预测达到目标所需时间
        double weeksNeeded = (targetScore - currentScore) / improvementRate;
        
        // 考虑边际效应（越高分越难提升）
        double difficultyFactor = calculateDifficultyFactor(currentScore, targetScore);
        weeksNeeded *= difficultyFactor;
        
        ProgressPrediction prediction = new ProgressPrediction();
        prediction.setCurrentScore(currentScore);
        prediction.setTargetScore(targetScore);
        prediction.setTargetPosition(targetPosition);
        prediction.setEstimatedWeeks((int) Math.ceil(weeksNeeded));
        prediction.setEstimatedDate(LocalDateTime.now().plusWeeks((long) weeksNeeded));
        prediction.setConfidence(calculateConfidence(trend));
        
        return prediction;
    }
    
    /**
     * 计算难度系数
     */
    private double calculateDifficultyFactor(double current, double target) {
        // 分数越高，提升越难
        // 60→70 相对容易，80→90 更难
        double gap = target - current;
        double baseFactor = 1.0;
        
        if (current >= 80) {
            baseFactor = 1.5;  // 80 分以上难度增加 50%
        }
        
        return baseFactor * (1 + gap / 100);
    }
}
```

```java
            // 分数提升超过 15 分视为跃升

                breakthroughs.add(new BreakthroughPoint(
                    currReport.getCreatedAt(),
                    prevReport.getOverallScore(),
                    currReport.getOverallScore()
                ));
            }
        }
```

```java
     * 验证教育时间线
     */
    private boolean validateEducationTimeline(List<Education> education) {
        for (int i = 0; i < education.size() - 1; i++) {
            Education current = education.get(i);
            Education next = education.get(i + 1);
            
            // 检查时间重叠
            if (current.getEndDate().isAfter(next.getStartDate())) {
                return false;
            }
        }
        return true;
    }
    
    /**
     * 识别时间冲突
     */
    private List<TimelineConflict> identifyConflicts(ParsedResume resume) {
        List<TimelineConflict> conflicts = new ArrayList<>();
        
        // 检查工作与教育时间是否冲突
        for (WorkExperience work : resume.getWorkExperience()) {
            for (Education edu : resume.getEducation()) {
                if (hasTimeOverlap(work, edu)) {
                    conflicts.add(new TimelineConflict(
                        "工作与教育时间重叠",
                        work.getCompany(),
                        edu.getSchool()
                    ));
                }
            }
        }
        
        // 检查工作之间是否冲突
        for (int i = 0; i < resume.getWorkExperience().size() - 1; i++) {
            WorkExperience current = resume.getWorkExperience().get(i);
            WorkExperience next = resume.getWorkExperience().get(i + 1);
            
            if (hasTimeOverlap(current, next)) {
                conflicts.add(new TimelineConflict(
                    "工作经历时间重叠",
                    current.getCompany(),
                    next.getCompany()
                ));
            }
        }
        
        return conflicts;
    }
```

```java
public class SkillMatchingService {

    private SkillNormalizationService normalizationService;

     * 对比简历技能与目标岗位要求的匹配程度
     */
    public SkillMatchResult calculateSkillMatch(
            List<NormalizedSkill> resumeSkills,
            String positionCode) {
        
        // 获取岗位要求技能
        List<RequiredSkill> requiredSkills = getRequiredSkills(positionCode);
        
        // 计算匹配度
        int matchedCount = 0;
        int requiredCount = requiredSkills.size();
        List<String> missingSkills = new ArrayList<>();
        List<String> extraSkills = new ArrayList<>();
        
        for (RequiredSkill required : requiredSkills) {
            boolean matched = resumeSkills.stream()
                .anyMatch(s -> s.getStandardName().equals(required.getSkillName()));
            
            if (matched) {
                matchedCount++;
            } else {
                missingSkills.add(required.getSkillName());
            }
        }
        
        // 找出简历中超出岗位要求的技能
        for (NormalizedSkill resumeSkill : resumeSkills) {
            if (requiredSkills.stream()
                    .noneMatch(r -> r.getSkillName().equals(resumeSkill.getStandardName()))) {
                extraSkills.add(resumeSkill.getStandardName());
            }
        }
        
        SkillMatchResult result = new SkillMatchResult();
        result.setMatchRate((double) matchedCount / requiredCount * 100);
        result.setMatchedSkills(matchedCount);
        result.setRequiredSkills(requiredCount);
        result.setMissingSkills(missingSkills);
        result.setExtraSkills(extraSkills);
        
        return result;
    }
}
```

```javascript
// 前端语音采集模块
class VoiceAnalyzer {
    constructor() {
        this.mediaRecorder = null;
        this.audioChunks = [];
        this.startTime = null;
        this.speechSegments = [];
        this.pauseSegments = [];
    }
    
    /**
     * 开始录音并实时分析
     */
    async startRecording() {
        const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
        this.mediaRecorder = new MediaRecorder(stream);
        this.mediaRecorder.start();
        
        this.startTime = Date.now();
        let lastSpeechState = false;
        let segmentStart = 0;
        
        // 实时语音活动检测
        const audioContext = new AudioContext();
        const analyser = audioContext.createAnalyser();
        const microphone = audioContext.createMediaStreamSource(stream);
        microphone.connect(analyser);
        
        const dataArray = new Uint8Array(analyser.frequencyBinCount);
        
        const detectSpeech = () => {
            analyser.getByteFrequencyData(dataArray);
            const volume = dataArray.reduce((a, b) => a + b) / dataArray.length;
            const isSpeaking = volume > 30;  // 音量阈值
            
            if (isSpeaking && !lastSpeechState) {
                segmentStart = Date.now() - this.startTime;
            } else if (!isSpeaking && lastSpeechState) {
                const segmentEnd = Date.now() - this.startTime;
                this.speechSegments.push({
                    start: segmentStart,
                    end: segmentEnd,
                    duration: segmentEnd - segmentStart
                });
            } else if (!isSpeaking && !lastSpeechState) {
                // 停顿
            }
            
            lastSpeechState = isSpeaking;
            requestAnimationFrame(detectSpeech);
        };
        
        detectSpeech();
        
        this.mediaRecorder.ondataavailable = (event) => {
            this.audioChunks.push(event.data);
        };
    }
    
    /**
     * 停止录音并获取分析结果
     */
    async stopRecording() {
        return new Promise((resolve) => {
            this.mediaRecorder.onstop = () => {
                const audioBlob = new Blob(this.audioChunks, { type: 'audio/wav' });
                const totalTime = Date.now() - this.startTime;
                
                // 计算语音特征
                const features = this.calculateFeatures(totalTime);
                
                resolve({
                    audioBlob: audioBlob,
                    features: features
                });
            };
            this.mediaRecorder.stop();
        });
    }
    
    /**
     * 计算语音特征
     */
    calculateFeatures(totalTime) {
        const totalSpeechTime = this.speechSegments.reduce(
            (sum, seg) => sum + seg.duration, 0);
        const speechCount = this.speechSegments.length;
        
        // 语速（字/分钟）- 需要结合转录文本
        const wordsPerMinute = (estimatedWordCount / totalSpeechTime) * 60000;
        
        // 停顿频率
        const pauseFrequency = speechCount / (totalTime / 60000);
        
        // 平均停顿时长
        const avgPauseDuration = totalTime - totalSpeechTime / speechCount;
        
        return {
            totalTime: totalTime,
            totalSpeechTime: totalSpeechTime,
            speechSegments: speechCount,
            wordsPerMinute: wordsPerMinute,
            pauseFrequency: pauseFrequency,
            avgPauseDuration: avgPauseDuration,
            speechRate: this.classifySpeechRate(wordsPerMinute),
            fluency: this.calculateFluency()
        };
    }
    
    classifySpeechRate(wpm) {
        if (wpm > 180) return 'FAST';
        if (wpm < 120) return 'SLOW';
        return 'NORMAL';
    }
    
    calculateFluency() {
        // 流畅度计算逻辑
        // 基于停顿频率和时长
        return 'GOOD';  // 简化
    }
}
```

```java
@Service
public class PauseAnalysisService {
    
    /**
     * 分析停顿频率和时长，判断思考深度和紧张程度
     */
    public PauseAnalysis analyzePausePattern(VoiceFeatures features) {
        PauseAnalysis analysis = new PauseAnalysis();
        
        // 停顿频率分析
        if (features.getPauseFrequency() > 20) {
            analysis.setFrequency(PauseFrequency.HIGH);
            analysis.addObservation("停顿频繁，可能表示紧张或思路不清晰");
        } else if (features.getPauseFrequency() < 8) {
            analysis.setFrequency(PauseFrequency.LOW);
            analysis.addObservation("停顿较少，表达流畅");
        } else {
            analysis.setFrequency(PauseFrequency.NORMAL);
        }
        
        // 停顿时长分析
        if (features.getAvgPauseDuration() > 3000) {
            analysis.setAvgDuration(PauseDuration.LONG);
            analysis.addObservation("停顿时间较长，可能在深度思考");
        } else if (features.getAvgPauseDuration() < 500) {
            analysis.setAvgDuration(PauseDuration.SHORT);
            analysis.addObservation("停顿时间短，可能准备充分或语速快");
        } else {
            analysis.setAvgDuration(PauseDuration.NORMAL);
        }
        
        // 停顿类型判断
        analysis.setPauseType(classifyPauseType(features));
        
        return analysis;
    }
    
    /**
     * 判断停顿类型
     */
    private PauseType classifyPauseType(VoiceFeatures features) {
        // 思考性停顿：停顿时间长，但频率适中
        if (features.getAvgPauseDuration() > 2000 && 
            features.getPauseFrequency() < 15) {
            return PauseType.THINKING;
        }
        
        // 紧张性停顿：停顿频繁且时长短
        if (features.getPauseFrequency() > 15 && 
            features.getAvgPauseDuration() < 1000) {
            return PauseType.NERVOUS;
        }
        
        // 正常停顿
        return PauseType.NORMAL;
    }
}
```

```java
@Service
public class ConfidenceAssessmentService {
    
    /**
     * 分析语气坚定程度和犹豫频率
     */
    public ConfidenceAssessment assessConfidence(
            VoiceFeatures features, 
            String transcript) {
        
        ConfidenceAssessment assessment = new ConfidenceAssessment();
        
        // 语音指标
        int confidenceScore = 50;  // 基础分
        
        // 语速适中加分
        if (features.getWordsPerMinute() >= 140 && 
            features.getWordsPerMinute() <= 160) {
            confidenceScore += 10;
        }
        
        // 停顿适中加分
        if (features.getPauseFrequency() >= 8 && 
            features.getPauseFrequency() <= 15) {
            confidenceScore += 10;
        }
        
        // 语调稳定加分
        if (features.getPitchVariance() >= 30 && 
            features.getPitchVariance() <= 100) {
            confidenceScore += 10;
        }
        
        // 语言指标
        if (transcript != null) {
            // 检测犹豫词
            int hesitationCount = countHesitationWords(transcript);
            if (hesitationCount > 10) {
                confidenceScore -= 20;
                assessment.addObservation("犹豫词较多（嗯、啊、这个...）");
            }
            
            // 检测确定性表达
            int certaintyCount = countCertaintyWords(transcript);
            if (certaintyCount > 5) {
                confidenceScore += 15;
                assessment.addObservation("使用较多确定性表达");
            }
        }
        
        assessment.setScore(Math.max(Math.min(confidenceScore, 100), 0));
        assessment.setLevel(classifyConfidenceLevel(assessment.getScore()));
        
        return assessment;
    }
    
    private int countHesitationWords(String transcript) {
        String[] hesitationWords = {"嗯", "啊", "呃", "这个", "那个", "就是", "然后"};
        int count = 0;
        for (String word : hesitationWords) {
            count += countOccurrences(transcript, word);
        }
        return count;
    }
    
    private int countCertaintyWords(String transcript) {
        String[] certaintyWords = {"肯定", "一定", "明确", "当然", "无疑", "确定"};
        int count = 0;
        for (String word : certaintyWords) {
            count += countOccurrences(transcript, word);
        }
        return count;
    }
}
```

```java
        LocalDateTime startDate = LocalDateTime.now().minusDays(days);
        List<UserFeedback> feedbacks = feedbackRepository.findByCategoryAndCreatedAtAfter(
            category, startDate);
        
        FeedbackTrend trend = new FeedbackTrend();
        trend.setCategory(category);
        trend.setTotalCount(feedbacks.size());
        trend.setAverageRating(feedbacks.stream()
            .mapToDouble(UserFeedback::getRating)
            .average()
            .orElse(0));
        trend.setCommonIssues(extractCommonIssues(feedbacks));
        trend.setSuggestions(extractSuggestions(feedbacks));
        
        return trend;
    }
    
    /**
     * 将反馈转化为改进任务
     */
    public ImprovementTask createTaskFromFeedback(Long feedbackId) {
        UserFeedback feedback = feedbackRepository.findById(feedbackId).orElseThrow();
        
        ImprovementTask task = new ImprovementTask();
        task.setTitle("处理用户反馈 #" + feedbackId);
        task.setDescription(feedback.getContent());
        task.setPriority(feedback.getRating() <= 2 ? "HIGH" : "MEDIUM");
        task.setCategory(feedback.getCategory());
        task.setStatus("OPEN");
        
        return improvementTaskRepository.save(task);
    }
}
```

```java
/**
 * 知识库定期更新机制
 */
@Service
public class KnowledgeBaseUpdateService {
    
    @Autowired
    private RagService ragService;
    
    @Autowired
    private WebCrawlerService webCrawlerService;
    
    // 技术趋势源
    private final List<String> trendSources = Arrays.asList(
        "https://github.com/trending",
        "https://news.ycombinator.com/",
        "https://www.infoq.com/"
    );
    
    /**
     * 定期更新知识库（每周执行）
     */
    @Scheduled(cron = "0 0 2 * * MON")
    @Transactional
    public void weeklyUpdate() {
        log.info("开始每周知识库更新");
        
        // 1. 爬取最新技术趋势
        List<TechTrend> trends = webCrawlerService.crawlTechTrends(trendSources);
        
        // 2. 识别与岗位相关的趋势
        for (String positionCode : Arrays.asList("backend", "frontend", "qa", "algorithm")) {
            List<TechTrend> relevantTrends = filterRelevantTrends(trends, positionCode);
            
            // 3. 添加到动态层
            for (TechTrend trend : relevantTrends) {
                addDynamicKnowledge(positionCode, trend);
            }
        }
        
        // 4. 清理过时知识
        cleanupOutdatedKnowledge();
        
        log.info("知识库更新完成");
    }
    
    /**
     * 添加动态知识
     */
    private void addDynamicKnowledge(String positionCode, TechTrend trend) {
        KnowledgeItem item = new KnowledgeItem();
        item.setPositionId(positionCode);
        item.setCategory("技术趋势");
        item.setTitle(trend.getTitle());
        item.setContent(trend.getSummary());
        item.setLayer("dynamic");
        item.setUpdatedAt(LocalDateTime.now());
        item.setSource(trend.getUrl());
        
        ragService.addKnowledgeItem(item);
    }
    
    /**
     * 清理过时知识
     */
    private void cleanupOutdatedKnowledge() {
        // 标记超过 2 年未更新的技术知识
        LocalDateTime cutoffDate = LocalDateTime.now().minusYears(2);
        List<KnowledgeItem> outdatedItems = knowledgeRepository.findByUpdatedAtBeforeAndLayer(
            cutoffDate, "basic");
        
        for (KnowledgeItem item : outdatedItems) {
            item.setDeprecated(true);
            knowledgeRepository.save(item);
        }
    }
}
```

## 总结

本优化方案涵盖七大核心模块的详细实现思路：

1. **打分逻辑模块**：五维评分模型、动态权重、对比学习
2. **RAG 模块**：四层知识库、多模态检索、质量保障
3. **提问模块**：动态生成、智能追问、多角色设定
4. **历史总结模块**：能力图谱、趋势分析、进步预测
5. **简历 PDF 模块**：智能解析、技能标准化、匹配度评估
6. **语音情绪模块**：特征提取、情绪识别、实时反馈
7. **综合优化**：数据中台、闭环反馈、个性化路径

实施建议按六个阶段进行，每阶段 2 周，总计 12 周完成全部功能。

关键成功因素：
- 大模型 API 的合理使用（混合架构控制成本）
- 数据质量保障（知识库持续更新）
- 用户体验优先（渐进式难度、成就系统）
- 持续迭代优化（A/B 测试、用户反馈闭环）

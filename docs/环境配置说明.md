# AI 模拟面试与能力提升软件 - 环境配置说明

## 一、系统要求

### 1.1 硬件要求
- CPU: 双核处理器及以上
- 内存：4GB 及以上
- 硬盘：1GB 可用空间
- 网络：需要互联网连接（用于调用阿里云 DashScope API）

### 1.2 软件要求
- JDK 17 或更高版本
- Maven 3.6 或更高版本
- 现代浏览器（Chrome 90+、Edge 90+ 等，用于语音输入功能）

## 二、环境搭建步骤

### 2.1 安装 JDK 17

**Windows 系统：**
1. 访问 Oracle 官网或 Adoptium 下载 JDK 17
2. 运行安装程序，按照提示完成安装
3. 配置环境变量：
   - 添加 `JAVA_HOME` 指向 JDK 安装目录
   - 在 `Path` 中添加 `%JAVA_HOME%\bin`
4. 验证安装：`java -version`

**Linux 系统：**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# CentOS/RHEL
sudo yum install java-17-openjdk-devel

# 验证安装
java -version
```

**macOS 系统：**
```bash
# 使用 Homebrew
brew install openjdk@17

# 或者从官网下载安装
```

### 2.2 安装 Maven

**Windows 系统：**
1. 访问 Maven 官网下载最新版本
2. 解压到指定目录
3. 配置环境变量：
   - 添加 `MAVEN_HOME` 指向 Maven 安装目录
   - 在 `Path` 中添加 `%MAVEN_HOME%\bin`
4. 验证安装：`mvn -version`

**Linux/macOS 系统：**
```bash
# Ubuntu/Debian
sudo apt install maven

# macOS (使用 Homebrew)
brew install maven

# 验证安装
mvn -version
```

### 2.3 获取阿里云 DashScope API Key

本项目使用阿里云 DashScope（通义千问）作为大模型和语音服务提供商：

1. 访问 [阿里云 DashScope 控制台](https://dashscope.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 完成实名认证（需要）
4. 在 `API Key 管理` 页面创建 API Key
5. 复制 API Key 备用

## 三、配置项目

### 3.1 配置文件位置

配置文件位于 `src/main/resources/application.yml`

### 3.2 配置 API Key

编辑 `application.yml`，将 `api-key` 替换为您的实际 API Key：

```yaml
app:
  llm:
    provider: dashscope
    api-key: sk-your-actual-api-key-here  # 替换为您的 API Key
    base-url: https://dashscope.aliyuncs.com/compatible-mode/v1
    model: qwen3.6-plus
```

### 3.3 完整配置说明

```yaml
# 服务器配置
server:
  port: 8080                    # 服务端口
  servlet:
    context-path: /api          # 上下文路径

# 应用配置
app:
  # 大模型配置
  llm:
    provider: dashscope         # API 提供商
    api-key: sk-xxxx            # API Key
    base-url: https://dashscope.aliyuncs.com/compatible-mode/v1
    model: qwen3.6-plus         # 默认模型
    enable-thinking: true       # 启用深度思考
    fallback-models:            # 备选模型
      - qwen3.6-plus
      - qwen-plus
      - qwen-max
  
  # RAG 配置
  rag:
    enable-vector-search: true  # 启用向量搜索
    vector-weight: 0.7          # 向量搜索权重
  
  # PDF 解析配置
  pdf:
    parser:
      tessdata-path: ./tessdata  # Tesseract 数据路径
      ocr-lang: chi_sim+eng      # 识别语言
  
  # 系统配置
  system:
    interview-timeout: 60       # 面试超时时间 (分钟)
    max-answer-length: 500      # 每轮回答最大字数
    max-history-records: 50     # 历史面试记录保留数量
```

## 四、编译和运行

### 4.1 编译项目

在项目根目录执行：
```bash
mvn clean package -DskipTests
```

### 4.2 运行项目

**方式一：使用 Maven**
```bash
mvn spring-boot:run
```

**方式二：运行打包后的 JAR**
```bash
java -jar target/interview-simulation-1.0.0.jar
```

**方式三：在 IDE 中运行**
- IntelliJ IDEA：右键 `Application.java` -> Run
- Eclipse：右键 `Application.java` -> Run As -> Java Application

### 4.3 访问应用

启动成功后，打开浏览器访问：
```
http://localhost:8080/api/
```

或者直接访问前端页面：
```
http://localhost:8080/api/index.html
```

## 七、常见问题

### 7.1 端口被占用

如果 8080 端口被占用，修改 `application.yml`：
```yaml
server:
  port: 8081  # 改为其他端口
```

### 7.2 API Key 无效

确保 API Key 正确配置，检查：
- API Key 是否正确复制
- 阿里云账号是否完成实名认证
- 是否有足够的 tokens 额度

### 7.3 数据库连接失败

项目使用 SQLite 数据库，数据文件位于 `./data/interview.db`
如果出现问题，可以删除该文件重新启动（会丢失数据）。

### 7.4 语音输入不可用

语音输入使用浏览器 Web Speech API，需要：
- 使用 Chrome、Edge 等支持的浏览器
- 允许浏览器访问麦克风
- 使用 HTTPS 或 localhost 访问

### 7.5 PDF 解析失败

- 确保 PDF 文件是文本型 PDF（非扫描图片）
- 如果是扫描版 PDF，需要配置 Tesseract OCR
- 下载中文语言包：https://github.com/tesseract-ocr/tessdata
- 将 `chi_sim.traineddata` 放入 `tessdata` 目录

### 7.6 向量搜索失败

向量搜索依赖阿里云 DashScope 的 embedding 服务：
- 确保 API Key 有效
- 确保网络连接正常
- 可在配置文件中禁用向量搜索：
  ```yaml
  app:
    rag:
      enable-vector-search: false
  ```
